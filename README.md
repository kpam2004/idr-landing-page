# 🌐 Institute of Digital Risk (IDR) Website

A modern, responsive homepage designed for the **Institute of Digital Risk (IDR)** — an industry-led training and deployment institute focused on digital, cyber, and AI risk.

---

## 🚀 Features

- ✅ Responsive design (Mobile + Desktop)
- ✅ Sticky navigation bar
- ✅ Smooth scrolling
- ✅ Modern dark UI with gradients
- ✅ Service cards (Academy, Innovation, Advisory)
- ✅ Contact form
- ✅ Clean and minimal design

---

## 🎨 Design Highlights

- Cube-based logo representing **risk, structure, and resilience**
- Color palette:
  - Orange → Innovation & urgency
  - Black → Authority & security
  - White → Clarity & simplicity
- Glassmorphism navbar for premium UI

---

## 🛠️ Tech Stack

- HTML5 (Semantic structure)
- CSS3 (Flexbox + Responsive design)
- JavaScript (Smooth scrolling)
